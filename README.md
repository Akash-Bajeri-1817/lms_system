# LMS Platform

A full-stack Learning Management System built with
Spring Boot 3, PostgreSQL, Elasticsearch, and AWS S3.

## Features
- JWT authentication with RBAC
- Course management with video lessons
- Razorpay payment integration
- Elasticsearch powered search
- Auto-grading quiz engine
- Progress tracking + certificates
- Email notifications

## Tech Stack
Java 21, Spring Boot 3.x, PostgreSQL,
Elasticsearch, Redis, Docker, AWS S3

## Run locally
docker-compose up
